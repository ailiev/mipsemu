Installing MIPS Emulator

Prerequisites

  • a C++ compiler.
  • log4cpp, version 3.5rc3 or later.
  • A dietlibc installation, for various MIPS-specific OS values, eg. syscall numbers. The location is pointed to by DIETLIBC_DIR in config.make, and we provide a dietlibc instance in BASEDIR/src/crossgcc/dietlibc/.

Configuration

Edit config.make as needed.

Building

$ make
$ make install
